# Customer-Churn-Analysis
📊 Customer Churn Analysis

📌 About This Project

In this project, I explored customer churn to understand why customers stop using a service and what factors influence their decision. The idea was to go beyond just numbers and actually find patterns that could help a business retain its customers.

I worked through the complete data analysis process — starting from raw data, cleaning it, exploring trends, and finally building machine learning models to predict churn.

🚀 What This Project Covers

Cleaning and preparing raw data
Exploring customer behavior through EDA
Visualizing trends and patterns
Building and testing machine learning models
Comparing models to find the best one

🛠️ Tools & Technologies

Python
Pandas, NumPy
Matplotlib, Seaborn
Scikit-learn
Jupyter Notebook

🔍 What I Did

Cleaned the dataset by handling missing values and fixing inconsistencies
Explored the data to understand how different features relate to churn
Created visualizations to make patterns easier to interpret
Converted categorical data into numerical form for modeling
Built multiple models like Logistic Regression, Decision Tree, and Random Forest
Evaluated each model using metrics like accuracy, precision, recall, and F1-score
Selected the best model based on performance

📊 Key Takeaways

Customers with shorter tenure were more likely to churn
Higher monthly charges often led to higher churn
Customers without long-term contracts tended to leave more
Certain usage patterns clearly influenced customer retention

📈 Why This Matters

This project shows how data can be used to make smarter business decisions. By identifying customers who are likely to churn, companies can take action early — like offering better plans or support — to retain them.

It also helped me understand how real-world problems can be approached using data science, not just from a technical perspective but also from a business point of view.

🧠 What I Learned

How to approach a real-world dataset from scratch
The importance of data cleaning and feature selection
How different machine learning models behave
How to turn analysis into meaningful insights
